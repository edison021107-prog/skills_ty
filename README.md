# OpenClaw Skills

个人 OpenClaw Skill 集合，用于扩展 AI 助手的能力。

## Skills 列表

| Skill | 描述 | 触发词 |
|-------|------|--------|
| `analyze-skill` | 从飞书表格生成 AI 模型评测标准化报告 | `/analyze` |
| `pm-skill` | 产品经理双视角分析（Senior PM 执行视角 + PM Leader 宏观视角） | `/pm` |

## 使用方法

1. 将 skill 目录复制到你的 OpenClaw workspace skills 目录
2. 重启 OpenClaw 或热加载技能
3. 使用对应的触发词调用
